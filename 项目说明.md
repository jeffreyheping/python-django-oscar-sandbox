# Django Oscar Sandbox 学习项目

这是一个专门用于学习 django-oscar 的独立项目，基于 GitHub 官方 sandbox。

## 项目结构

```
django-oscar/
├── apps/             # 自定义应用
│   └── user/         # 用户扩展模型
├── fixtures/         # 测试数据
│   ├── *.csv        # 产品数据
│   ├── *.json       # 配置数据
│   └── images.tar.gz # 产品图片
├── public/           # 静态文件和媒体文件（运行时生成）
├── static/           # 项目静态资源
├── __init__.py
├── manage.py         # Django 管理脚本
├── settings.py       # 项目配置
├── urls.py           # URL 路由
└── wsgi.py           # WSGI 配置
```

## 完整安装步骤（从零开始）

### 1. 安装 Django Oscar

```bash
pip install django-oscar[sorl-thumbnail]
```

> 官方文档推荐使用 pip 安装标准版，不推荐使用开发模式安装。

### 2. 创建项目目录并复制文件

```bash
# 从 GitHub 下载 sandbox 目录
# https://github.com/django-oscar/django-oscar/tree/master/sandbox
```

所需文件：
- `__init__.py`
- `manage.py`
- `settings.py`
- `urls.py`
- `wsgi.py`
- `apps/` 目录
- `fixtures/` 目录
- `static/` 目录

### 3. 数据库迁移

```bash
python manage.py migrate
```

### 4. 收集静态文件

```bash
python manage.py collectstatic
```

### 5. 导入产品数据（可选）

如果需要导入示例产品数据：

```bash
# 导入产品数据
python manage.py oscar_import_catalogue --flush fixtures/books.essential.csv

# 解压产品图片
tar -xzf fixtures/images.tar.gz -C public/media/

# 自动关联图片到产品（重要！官方步骤）
python manage.py oscar_import_catalogue_images public/media --filename upc
```

### 6. 加载其他测试数据（可选）

```bash
python manage.py loaddata fixtures/*.json
```

### 7. 创建管理员账户

```bash
python manage.py createsuperuser
```

或者在 Python Shell 中创建：

```bash
python manage.py shell
```

```python
from django.contrib.auth import get_user_model
User = get_user_model()
User.objects.create_superuser('admin', 'admin@example.com', 'admin')
```

### 8. 启动开发服务器

```bash
python manage.py runserver 8000
```

## 重要说明

### 项目独立性

✅ 这个项目是**完全独立的**！
- 不需要本地的 django-oscar 源码
- 通过 pip 安装的 django-oscar 包运行
- 可以在任何地方部署

### 运行时生成的文件

这些文件在首次运行时会自动生成，可以安全删除和重建：

| 文件/目录 | 删除后重建命令 | 说明 |
|---------|---------------|------|
| `public/static/` | `python manage.py collectstatic` | Django 静态文件集合 |
| `public/media/` | `tar -xzf fixtures/images.tar.gz -C public/media/` | 产品图片目录 |
| `db.sqlite` | `python manage.py migrate` | 数据库文件 |
| `whoosh_index/` | `python manage.py update_index` | 搜索引擎索引 |

### 图片关联

⚠️ **重要发现**：Oscar 官方要求手动运行图片关联命令！

CSV 导入只会导入产品数据，不会自动关联图片。必须运行：

```bash
python manage.py oscar_import_catalogue_images public/media --filename upc
```

## 常用命令

```bash
# 进入 Django Shell
python manage.py shell

# 创建数据迁移
python manage.py makemigrations

# 执行数据库迁移
python manage.py migrate

# 收集静态文件
python manage.py collectstatic

# 重建搜索引擎索引
python manage.py update_index

# 清理搜索引擎索引
python manage.py clear_index

# 导入产品数据
python manage.py oscar_import_catalogue fixtures/books.essential.csv

# 导入产品图片
python manage.py oscar_import_catalogue_images public/media --filename upc
```

## 故障排除

### 图片不显示

1. 检查 `public/media/` 目录是否有图片文件
2. 检查数据库中产品是否关联了图片：`python manage.py shell` 后查询
3. 运行图片关联命令：`python manage.py oscar_import_catalogue_images public/media --filename upc`

### 搜索功能不工作

```bash
python manage.py clear_index
python manage.py update_index
```

### 数据库错误

删除 `db.sqlite` 后重新迁移：

```bash
rm db.sqlite
python manage.py migrate
```
